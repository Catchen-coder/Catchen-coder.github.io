Python爬虫爬取热门网络小说

发表于 2019-10-07 | 分类于 Python | 阅读数 次

前言

相信很多人和我遇到过相同的问题，目前主流的小说网站中小说价格比较昂贵，但是盗版小说网站中的广告又比较多。而且目前提供TXT小说下载的网站越来越少。所以我制作了这款网络爬虫，可以整本爬取笔趣阁中的小说。

网络爬虫

网络爬虫（又称为网页蜘蛛，网络机器人，在FOAF社区中间，更经常的称为网页追逐者），是一种按照一定的规则，自动地抓取万维网信息的程序或者脚本。另外一些不常使用的名字还有蚂蚁、自动索引、模拟程序或者蠕虫。[1]

运行环境

本人Python版本为3.7.4

程序运行需要调用bs4包,可以通过命令
pip install bs4
来进行安装，但是因为pip默认的源在国内实在是太慢，所以我建议各位小伙伴使用豆瓣镜像源来进行安装。使用豆瓣镜像源的命令为：

pip install -i https://pypi.douban.com/simple bs4。

没错，使用豆瓣源安装程序的命令仅仅是在pip install后面加上-i https://pypi.douban.com/simple。其中参数-i代表指向一个储存库。而https://pypi.douban.com/simple/就是国内的豆瓣python包储存库啦。

运行程序

在运行程序前还需要对程序中一些内容进行修改。

  1. 修改起始URL，将起始URL修改为需要爬取的小说的第一章URL。
  2. 修改结束URL，同起始URL类似，需要将结束URL修改为最后一章的URL。程序在爬取完成这个URL后将会自动结束。
  3. 修改文件URL，需要将文件URL修改为想保存的位置以及文件名，程序会试图打开这个文件，如果文件不存在，程序将会自动建立这个文件。
  4. 开始运行。
  5. 程序运行完成后就可以在目标文件夹中看到已经下载好的TXT格式小说啦。
